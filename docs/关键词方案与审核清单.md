# 关键词方案与审核清单

## 小程序名称建议

按优先级排列：

| 名称 | 关键词覆盖 | 审核风险 | 推荐 |
|------|-----------|---------|------|
| 性格解码器 | 性格、解码 | ✅ 低 | 🥇 首选 |
| 性格密码 | 性格、密码 | ✅ 低 | 🥈 |
| 深度性格画像 | 性格、画像 | ✅ 低 | 🥉 |
| 人格探测器 | 人格、探测 | 🟡 中 | 备选 |
| 性格匹配测试 | 性格、测试、匹配 | ✅ 低 | 备选 |

**名称规则**：
- ✅ 含核心关键词："性格""人格""解码""密码""测试"
- ✅ 不含"AI""智能""大模型""人工智能"
- ✅ 4-8 个字，好记
- ❌ 不含"MBTI"（商标风险）
- ❌ 不含"心理医生""心理咨询"（需要资质）

## 审核素材清单

### 小程序图标
- [ ] 1024x1024 方形图标
- [ ] 不含"AI"字眼
- [ ] 简洁抽象，建议用几何图形/渐变

### 审核截图（至少 5 张）
- [ ] 首页截图
- [ ] 答题页截图（题目 + 选项）
- [ ] 结果页 - 性格类型
- [ ] 结果页 - 维度分析
- [ ] 分享卡片预览

**截图检查**：所有截图中出现的文字都不包含"AI""人工智能""AI 生成"。

### 类目选择
- [ ] 一级类目：**工具**
- [ ] 二级类目：**信息查询**

### 小程序描述（提交用）
```
性格解码器是一款基于心理学理论和大数据分析的
个人性格画像工具。通过5道精选题，系统为您匹
配专属的性格分析报告，帮助您深入了解自己的
隐藏特质和行为模式。支持分享性格卡片，看看
您和朋友的性格匹配度。
```

### 提交前最终检查
- [ ] 小程序名称不含"AI"
- [ ] 简介/描述不含"AI"/"人工智能"/"大模型"
- [ ] 截图不含"AI"字眼
- [ ] 类目选"工具 - 信息查询"
- [ ] 不涉及社交（交友/匹配/附近的人）
- [ ] 不涉及医疗（诊断/治疗/咨询建议）
- [ ] 有完整的免责声明："仅供娱乐参考，不构成专业心理评估"
- [ ] iOS 端不展示付费入口
- [ ] 隐私政策页面（小程序内）
- [ ] 用户协议页面（小程序内）

## Mini 版 vs 完整版对照

| 内容 | Mini 版（Day 1 提交） | 完整版（Day 5 提交） |
|------|---------------------|---------------------|
| 题目数量 | 5 题 | 30 题 |
| 结果生成 | 静态模板（4 种） | 云函数 + 混元 API |
| 免费维度 | 3 个 | 3 个 |
| 付费维度 | 无 | 4 个 + 附加内容 |
| 支付 | 不涉及 | 微信支付（仅 Android） |
| 分享卡片 | 内置模板 | Canvas 动态生成 |
| 审核通过率 | 高（功能简单） | 中（含支付） |

## 常见拒审原因及预防

| 拒审原因 | 预防措施 |
|---------|---------|
| 类目选择错误 | 确保选"工具 - 信息查询" |
| 涉及 AI 生成内容 | 全文避开 AI 字眼 |
| 功能不完整 | Mini 版也要完成答题→结果完整闭环 |
| 涉及社交 | 不出现"交友""匹配异性"等 |
| 涉及医疗/健康 | 不出现"诊断""治疗""心理医生" |
| iOS 虚拟支付 | iOS 端不展示付费入口 |
| 隐私政策缺失 | 内置隐私政策和免责声明 |
| 截图与真实内容不符 | 截图必须是小程序内的真实页面 |
